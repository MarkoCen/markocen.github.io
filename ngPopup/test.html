Hello! This is ngPopup, a super-easy-use AngularJS dialog, you can test some of its functions here.

ngPopup is jQuery-independent, it only rely on AngularJS, and it supports angular two-way binding. Try dragging dialog title bar or slider to see the data binding.

Also, after initialization, you can call ngPopup methods in controller. Try clicking open/close button which would call $scope.myDialog.open()/$scope.myDialog.close().

ngPopup supports two ways to embed contents, template or templateUrl, exactly the same way as AngularJS directive did.
